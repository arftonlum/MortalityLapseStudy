# Mortality Experience Study

This project is a personal actuarial project designed to analyze life insurance mortality and lapse experience using Python.

## Goals

- Clean and explore insurance data
- Compare observed mortality to an industry mortality table
- Calculate Actual-to-Expected (A/E) ratios
- Analyze lapse experience
- Produce visualizations and a professional actuarial report

## Tools

- Python
- pandas
- NumPy
- Matplotlib
- Git
- GitHubgit
